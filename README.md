### Prerequisites
* **Node.js** (v20 or higher)
* **npm**

### Installation
```bash
git clone https://github.com/dereklee0310/SULFUR-lab.git
cd SULFUR-lab
npm install
```

### Development
```
npm run dev -- --open
```
